Setting Up

Back to top

Hello, and thank you for purchasing my theme, Statix Landing. I hope you enjoy using it!

To start, extract the package and upload its contents to your server. Index.html is the template file and should be functional without further setup.

Index.php is a php version of index.html with the color swapper sidebar (for demo purposes). The other php and txt files included are used by the subscription and contact forms.

The contact form will need to modified so that it sends emails to the correct address, more details about that here.

Ensure that PHP is supported by your server, or else the contact form and subscription form scripts won't work.

The dummy text included in the template is hard-coded into the page. Change them at your leisure.

Color Schemes

Statix Landing comes with five predefined color schemes and a light/dark option. The color scheme can be selected by changing the line:

<link rel="stylesheet" href="css/theme-blue.css">

Change theme-blue.css to one of the following:

  • theme-blue.css
  • theme-teal.css
  • theme-green.css
  • theme-red.css
  • theme-orange.css

The light and dark theme for the main content body of the page can be chosen by changing the line <div class="container white-bg content">. white-bg can be swapped for black-bg, which enables the darker template.


Bootstrap

Statix Landing is built with the Bootstrap css framework. Statix Landing uses Bootstrap's 12-column responsive grid. The main.css stylesheet overrides some of the default styles included in the original Bootstrap package. Learn about Bootstrap here.


JavaScript

JavaScript is used in a few spots to style the page.

The slider images are vertically aligned by the equalize_slider() function.

Feature boxes don't look good if they aren't all the same height in a row. They can be forced into the same height with equalize_divs() so that their bottom borders align horizontally. This can be enabled by giving the row-fluid element a class of js-same-height. For example <div class="row-fluid js-same-height">

Images and their hover features are handled by JavaScript, not CSS.

Grid

This theme uses the 12-column grid by Bootstrap extensively. In a nutshell, create a row using <div class="row-fluid">, then create a div of width x columns with <div class="spanx"> where x is the number of columns to fill (range 1 to 12). Learn more here.

This theme wraps the Bootstrap row-fluid class with a row-padding class. This is simply for padding the row and it is recommended that you use it to surround all rows and content.

A typical hierarchy for a row should look as follows:

				
<!-- Ribbon -->
<div class="ribbon aligncenter">
     		<h2>title</h2>
     		<p class="strong">caption</p>
     	</div>
<!-- Row -->
<div class="row-padding">
    <div class="row-fluid">
		<div class="span12">	<!-- Or multiple spans -->
			<div class="feature-box"> <!-- Or box for no bg -->
				<h2>subtitle</h2>
				<!-- content -->
				<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.
				</p>
			</div>
		</div>
	</div>
</div>

Ribbons

The ribbon headings are a custom component of this theme. Simply add a <div class="ribbon"></div"> inside the div you want to overlay and it will fold around its parent. JavaScript takes care of styling the ribbons by appending extra components to it (such as the background folds).

Ribbons will only work on divs with 0px horizontal margin and padding, such as the main container classes.

h1 or h2 headings should be used in the ribbons.

All ribbons are created with CSS3, including large main header ribbon of the site. This means that changing their colors can be done by simply changing the style sheet (as seen in the different CSS files for the different color schemes).

Feature Boxes

This theme divides content into boxes and wraps the content into nice shadowed boxes for styling and layout padding.

Use the box class for a layout-only box (no background or shadow), and use the feature-box class to wrap content in a stylish shadowed box.

Slider

This theme users Flexslider as the main slider. See details here.

Slide content is inserted into: <div class="flexslider"> <ul class="slides"> <li> <!-- content here --> </li> </ul> </div>

This theme's default slider content divides data into left and right sections using <div class="slider-text"> and <div class="slider-image">. They take up 50% of the slider each by default. The images are vertically aligned by JavaScript in the slider by default.

Change the image sources to your liking to change the images in the slider as well as in the components on the rest of the page.

A <div class="container"> also wraps the content for extra padding.


Lightbox

This theme uses Lightbox for image popups. Read simple documentation here.

Wrap an img thumbnail element inside of an anchor link, and link it to the full sized image. Add rel="lightbox" to the anchor tag and you are good to go. Example: <a href="img/sample4.jpg" rel="lightbox"><img src="img/sample4_sm.jpg"/></a>

You can also group images together into a gallery by adding a group name inside square brackets to the rel attribute like so: rel="lightbox[groupname]".

You can add a description by adding a title attribute with the value set to the description in the anchor link, such as title="my caption".



Elastislide Gallery

This theme uses a modified version of the Elastislide gallery. See documentation here.

The sample gallery incorporates the aforementioned hover class and lightbox functionality. This gives it the hover effect and the lightbox popup.



Testimonials

Testimonials are placed in an instance of Flexslider. The image for each is automatically rounded and floated to the left side of the testimonial.

The quotes should be placed inside <p> tags and the names should be placed in <small> tags.

The image should be placed before the quote, and should have 1:1 dimensions (it will be resized to 100px by 100px).



Contact Form

Back to top

The theme's contact form uses Ajax to send the user's details in an email to you. You will need to change the default email address by changing send_email.php.

Change $contact_email = 'bobbyli68@gmail.com' from my email address to your own (keep it wrapped in the single quotes).

After this, the email form should be fully functional. It will send you basic plain-text emails with the information that the users enter. You can modify the script to give style the email itself if you'd like.

PHP must be installed on your server for this to work. The script uses the PHP mail function. More info here.

Users can access the contact page by the small email icons in the header and footer, or by the large 'Email Us' button in the footer.



Subscriber Form

Back to top

The theme's subscribe form uses Ajax to add user emails to a .txt file by default. This requires no set up, simply check the latest version of newsletter_emails.txt on your server to see the list of emails that have been entered.

MailChimp Integration

This theme's subscribe form in the footer is styled to support MailChimp's embed code. Create your embedded form in MailChimp (List -> For Your Website -> Super Slim Form).

Copy the provided HTML code. You only need everything inside and including the form tag, the outside divs and styles should not be added to this theme.

Finally, replace the form with the MailChimp form and you are done.
